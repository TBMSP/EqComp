# EqComp
EqComp es una calculadora sísmica para el calculo y estimación rápida sobre magnitud y epicentro de un evento sísmico de forma manual, permite la visualización de sismogramas y un mapa donde se dibujan las estaciones sísmicas con sus respectivas circunferencias radiales del posible epicentro en kilometros.

Para obtener los nombres de las redes sísmicas con sus respectivas estaciones puedes visitar la pagina de <a href="http://www.fdsn.org/networks/">FDSN</a>, para luego ingresarlas en el programa.

<br>

<b>Los parametros solicitados para cargar una estación sísmica al programa son los siguientes:</b>


<b>Centro de Datos (DataCenter)</b>:<br>
Se refiere al centro de datos encargada de almacenar estos datos, como <a href="https://www.iris.edu/hq/">Iris</a> o <a href="https://geofon.gfz-potsdam.de/">Geofon</a>.

<b>Red (Network)</b>:<br>
Se refiere a una red de estaciones sísmicas de algún operador ubicadas en distintas zonas de algún centro de datos anteriormente seleccionado.

<b>Estación (Station)</b>:<br>
Se refiere a una estación sísmica ubicada en algún lugar físico de alguna red sísmica anteriormente seleccionada.

<b>Fecha y hora en UTC (StartTime)</b>:<br>
Se refiere a la fecha y hora en formato UTC en que se quieren comenzar a obtener los datos de la estación anteriormente seleccionada.

<b>Duración en segundos de datos (Duration)</b>:<br>
Se refiere a la duración en segundos de datos a partir de la hora indicada anteriormente en "Fecha y Hora UTC".

<b>Canal de estación (Duration)</b>:
Se refiere a un canal de frecuencias donde la estación anteriormente seleccionada esta transmitiendo sus datos.

<b>Ubicación de semilla de estación (Duration)</b>:
Se refiere a un identificador de la semilla (o ubicación) de datos almacenados de la estación anteriormente seleccionada.
